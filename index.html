Spring 2026 · Electromagnetics

Half-Wave Dipole Antenna Simulation

Designed and simulated a half-wave dipole antenna in CST Studio Suite scaled to a custom operating frequency. Iteratively tuned geometry to meet S11 return loss specs. Constructed a layered human body model using custom dielectric materials (skin εr=41.32, fat εr=5.46, muscle εr=54.97) and analyzed antenna radiation patterns in three configurations: free space, on-body, and embedded inside muscle using Boolean subtraction.

CST Studio Suite RF Simulation S-Parameters EM Modeling

Designed and simulated a quarter-wave micro-strip impedance matching network on a flexible polyamide substrate using CST Studio Suite. Evaluated electromagnetic performance under mechanical bending conditions using S-parameter analysis.
